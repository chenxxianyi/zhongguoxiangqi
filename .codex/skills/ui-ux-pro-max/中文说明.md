# UI UX Pro Max 中文说明

## 这个 skill 是什么

这是一个 UI/UX 设计知识库，包含多种设计风格、配色、字体组合、UX 指南和图表类型建议，可用于网页和移动应用设计。

## 什么时候使用

- 用户要求设计、优化或重做 UI
- 需要选择视觉风格、配色或字体
- 需要改进交互体验
- 需要为具体行业或产品类型匹配设计方案

## 怎么使用

1. 先分析产品类型、行业、用户和目标场景。
2. 从知识库中匹配设计风格、配色和字体。
3. 将建议落实到组件、布局和交互细节。
4. 对最终页面做可用性和视觉一致性检查。

## 注意事项

- 不要只堆视觉效果，优先服务用户任务。
- 后台、SaaS、CRM 类产品应保持克制和高信息密度。
- 设计建议要结合项目已有设计系统。
