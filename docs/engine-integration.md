# 引擎集成

## 内置引擎

`internal/engine/builtin` 实现迭代加深 Negamax/Alpha-Beta，具备：

- 合法着生成和结果二次校验；
- `context.Context` 取消；
- 时间、深度、节点预算；
- 简单子力与兵卒推进评估；
- 捕获优先和稳定字典序，固定输入可复现；
- MultiPV 根候选结构。

它是可运行基础引擎，不宣称具有专业引擎棋力或精确 Elo。

## 外部引擎

Pikafish 等外部引擎必须作为用户/部署方自行安装的独立进程接入。计划中的 UCI/UCCI
适配器需包含握手、超时、取消、输出限制、进程池、崩溃恢复和最终走法规则复验。

本仓库当前没有提交、下载或分发任何第三方引擎二进制或 NNUE 文件。
