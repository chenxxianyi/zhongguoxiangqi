# 中国象棋规则与坐标约定

## 棋盘

- 内部棋盘为 `file 0..8`、`rank 0..9`；
- 内部 `rank 0` 是黑方底线，`rank 9` 是红方底线；
- 对外 ICCS 坐标以红方视角表示：`a0` 是红方左下角，`i9` 是黑方右上角；
- 坐标不会因为玩家执黑而翻转。

## FEN

采用常见中国象棋 FEN 棋子字符：

- `k/a/e/h/r/c/p`：黑方将、士、象、马、车、炮、卒；
- 大写字符表示红方；
- `w` 或 `r` 表示红方行棋，`b` 表示黑方行棋；
- API 规范输出当前使用两字段形式：`board side`。

初始局面：

```text
rheakaehr/9/1c5c1/p1p1p1p1p/9/9/P1P1P1P1P/1C5C1/9/RHEAKAEHR w
```

## 已实现裁决

- 所有棋子移动、阻挡和吃子规则；
- 马腿、象眼、象不过河、九宫限制；
- 炮架、将帅照面、自陷将军；
- 将军、将死、无合法着判负；
- MVP 休闲规则：同一哈希局面第三次出现判和；
- 认输和双方同意和棋属于业务终局。

长将、长捉等正式竞赛完整裁决尚未宣称实现，后续应通过版本化 `RuleSet` 增加。
