# 棋境 H5 UI Demo

本目录是依据 `前端开发方案.md` 与 `.codex/skills/ui-ux-pro-max` 设计并实现的纯 HTML/CSS/JavaScript 交互原型。

## 运行

可直接双击 `index.html`，或在项目根目录启动静态服务器：

```powershell
python -m http.server 4173 -d h5-demo
```

然后访问 `http://127.0.0.1:4173/`。

## 页面

- 首页
- 新对局设置
- 对弈棋盘
- 棋谱库
- 学习中心
- 复盘分析
- 历史对局
- 设置与诊断

## 可体验交互

- 侧边导航和移动端抽屉
- 深浅主题切换
- 执色、难度、AI 模式选择
- 棋子选择、落点提示、演示走子和翻转棋盘
- 着法/局面页签
- 棋谱文件选择与导入进度模拟
- 学习版本构建进度模拟
- 复盘、历史和设置页签

## 说明

这是视觉和交互 Demo，不包含完整中国象棋规则、真实 AI 搜索、棋谱解析或后端接口。正式 Vue 3 版本中，规则与对局状态必须由服务端作为权威来源。
